W3C是什么？

  1. W3C是万维网联盟（World Wide Web Consortium） 的简称；
  2. W3C这个组织创建于1994年10月；
  3. W3C的创建者是生于伦敦的蒂姆·伯纳斯一李（Tim Berners-Lee）爵士；
  4. W3C的日常工作是创建并维护WWW标准；
  5. W3C 是一个会员组织
  6. W3C 的工作是对 web 进行标准化
  7. W3C 标准被称为 W3C 推荐（W3C Recommendations）

w3c标准分类

对应的标准也分三方面：结构化标准语言主要包括XHTML和XML，表现标准语言主要包括CSS，行为标准主要包括对象模型（如W3C DOM）、ECMAScript等。 这些标准大部分由W3C起草和发布，也有一些是其他标准组织制订的标准，比如ECMA（European Computer Manufacturers Association）的ECMAScript标准。

结构标准

  1. XML
  2. XHTML

表现语言

  1. css

行为标准

  1. DOM
  2. ECMAScript
